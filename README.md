Features

Add Tasks: Add new tasks with title, due date, priority, and category.

Edit Tasks Inline: Click on a task title to edit it directly.

Toggle Completion: Mark tasks as complete or pending.

Filter & Sort: Filter tasks by status (Pending / Completed / All) and sort by due date.

Search: Quickly find tasks by title or category.

Persistent Storage: Tasks are stored in localStorage to persist across sessions.

Task Badges: Visual indicators for priority (Low, Medium, High) and category (Work, Personal, etc.).

Clear Storage: Reset all tasks with a single button.

Responsive Design: Fully responsive layout for mobile, tablet, and desktop.

Accessible: Focus rings and ARIA-friendly interactions.
